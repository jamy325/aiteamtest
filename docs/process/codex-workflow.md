# Codex Workflow

## Source of truth

Tasks come from GitHub Issues / AI Dev Team Board, not from local task documents.

## Task selection

Preferred flow:

1. Query GitHub task list.
2. Select one Ready P0 task.
3. Confirm prerequisites are completed.
4. Read the full Issue detail.
5. Implement only that Issue.
6. Create PR.
7. Comment PR URL and test results back to the Issue.
8. Wait for PM/review workflow.

## If GitHub task list is unavailable

Ask the user for a specific Issue number.
Do not guess the next task from local docs.
